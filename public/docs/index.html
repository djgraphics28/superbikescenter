Introduction

Base URL: https://superbikescentral.online

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Addresses

Get all provinces This endpoint retrieves all provinces.

Example request:
curl --request GET \
    --get "https://superbikescentral.online/api/provinces" \
    --header "Content-Type: application/json" \

const url = new URL(
    "https://superbikescentral.online/api/provinces"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",

Submit a new application This endpoint allows a user to submit a new application.

Example request:
curl --request POST \
    "https://superbikescentral.online/api/application" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"customer_id\": 1,
    \"product\": 2,
    \"source_of_income\": \"Employment\",
    \"name_of_business\": \"My Business\",
    \"company_name\": \"My Company\",
    \"work_position\": \"Manager\",
    \"years_in_work\": 5,
    \"comaker_name\": \"John Doe\",
    \"comaker_work\": \"Engineer\",
    \"comaker_monthly_income\": 50000,
    \"months_to_pay\": 12,
    \"monthly_payment_amount\": 1000,
    \"monthly_income\": 2000,
    \"downpayment\": 500,
    \"loan_amount\": 10000,
    \"name\": \"Jane Doe\",
    \"email\": \"jane@example.com\"
}"

const url = new URL(
    "https://superbikescentral.online/api/application"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "customer_id": 1,
    "product": 2,
    "source_of_income": "Employment",
    "name_of_business": "My Business",
    "company_name": "My Company",
    "work_position": "Manager",
    "years_in_work": 5,
    "comaker_name": "John Doe",
    "comaker_work": "Engineer",

    "monthly_income": 2000,
    "downpayment": 500,
    "loan_amount": 10000,
    "name": "Jane Doe",
    "email": "jane@example.com"

Change Password This endpoint allows a user to change their password.

Example request:
curl --request POST \
    "https://superbikescentral.online/api/change-password" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"current_password\": \"old_password\",
    \"new_password\": \"new_password123\",
    \"new_password_confirmation\": \"new_password123\"
}"

const url = new URL(
    "https://superbikescentral.online/api/change-password"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "current_password": "old_password",
    "new_password": "new_password123",
    "new_password_confirmation": "new_password123"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "message": "Password changed successfully."
}

Example response (400):


{
    "error": "Current password is incorrect."
}

Example response (422):


{
    "error": "The new password confirmation does not match."
